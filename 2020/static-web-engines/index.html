Static Web Engines

Written by Thai Ho with ♥ on 9 May 2020 in Web Development 1 min

For personal blogs

  • Hugo
  • Jekyll

For documentations

  • Docusaurus: open-sourced, good support by Facebook, good UI for navigation
  • Readthedocs: ideal for books, like this Python book
  • Gitbook: appropriate for team documentations
  • Bookstack: idem, open-sourced and self-hosting
  • Docsify: lightweight team documentation
  • Sphinx: API docs with simplified themes, i.e. Alabaster

For APIs

  • Swagger
  • [Doxygen]: for example, DBus
  • [Redoc]: for example, Docker
  • Postman
