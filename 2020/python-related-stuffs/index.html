Python-related stuff

Written by Thai Ho with ♥ on 16 April 2020 in 1 min

As a fan of Javascript, the first time I worked with Python gave me quite a lot of surprises. People often talk about Python as an easy and friendly language to learn. I, on the other hand, see the contradiction. It’s difficult to learn for me, mainly due to its strictly loose typing.

Here I summarize all difficulties I encountered during the use of Python. This is not an exhaustive list.

Serialization

Not every objects in Python are serializable. Dumping an object to JSON for logging purpose without paying attention will result in unexpected error, slowing down the development progress. It’s even worse if people heavily use Docker for the development purpose. Building a Docker container is not a pleasant experience.
